5. HTML Links - Hyperlinks

<a>...</a> - Defines a hyperlink, which is used to link from one page to another.

The basic structure of a hyperlink is an <a > tag with an `href` attribute pointing to the desired URL. The text between the opening and closing tags is the clickable part.

<a href="https://www.example.com">Visit Example</a>
Visit Example

Link Attributes

1. `href` - Specifies the URL of the page the link goes to.

<a href="https://www.wikipedia.org">Visit Wikipedia</a>
Visit Wikipedia

2. `target` - Defines where to open the linked document. Common values:

  • _self - Opens in the same tab (default behavior).
  • _blank - Opens in a new tab or window.
  • _parent - Opens the document in the parent frame.
  • _top - Opens the document in the full body of the window.
<a href="https://www.google.com" target="_blank">Open in New Tab</a>
Open in New Tab

3. `title` - Provides additional information about the link. The text appears when the user hovers over the link.

<a href="https://www.example.com" title="Example Website">Hover for Title</a>
Hover for Title

4. Email Links - Use mailto: in the href attribute to open the user's default email program.

<a href="mailto:someone@example.com">Send Email</a>
Send Email

5. Telephone Links - Use tel: in the href attribute to open the phone dialer on mobile devices.

<a href="tel:+201227315759">Call Us</a>
Call Us

Button as a Link

You can use a button as a link by adding some JavaScript to the onclick event. This event will trigger the specified action when the button is clicked, such as navigating to a different page or executing other JavaScript functions.

Example

In this example, the button will redirect the user to the specified URL when clicked:

6. HTML Images

<img> - Defines an image in an HTML document.

The <img> tag does not have a closing tag. Instead, it uses attributes to specify the image source and other properties.

<img src="image.jpg" alt="Description of Image">

Image Attributes

1. `src` - Specifies the path to the image file.

<img src="https://via.placeholder.com/150" alt="Sample Image">

2. `alt` - Provides alternative text for the image if it cannot be displayed. It is also important for accessibility purposes.

<img src="image.jpg" alt="A beautiful sunset">

3. `width` and `height` - Specifies the width and height of the image. The values can be in pixels or percentages.

<img src="image.jpg" width="300" height="200" alt="Sized Image">

4. `title` - Provides additional information about the image. It appears when the user hovers over the image.

<img src="image.jpg" alt="Sample Image" title="This is a sample image">

Images in Another Folder

When you store images in a separate folder, you need to reference them with the correct file path. The src attribute will include the folder name in the path to the image file.

Example of Storing Images in a Subfolder

Let's assume you have an images folder in the same directory as your HTML file. To reference an image inside that folder, you can use the relative path like this:

<img src="images/photo.jpg" alt="A photo from a folder">

Example with a Folder Inside Another Folder

You can also have images inside a folder that is inside another folder. The path will be relative to the location of the HTML file.

<img src="assets/images/photo.jpg" alt="Image in nested folder">

Using Absolute Paths

If the images are located outside the current project folder or on a different server, you can use an absolute path (full URL) to reference them.

<img src="https://www.example.com/images/photo.jpg" alt="Image from external server">

7. Using CSS

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements.
  • Internal - by using a <style> element in the <head> section.
  • External - by using a <link> element to link to an external CSS file.

1. Inline CSS

Inline CSS is used by adding the style attribute directly to HTML elements. It is usually used for quick styling of individual elements.

<p style="color: red; font-size: 20px;">This is a red colored text.</p>

This is a red colored text.

2. Internal CSS

Internal CSS is used by placing styles within a <style> element inside the <head> section. It applies styles to the entire document.

<style>
  body {
    color: #333;
    font-family: Arial, sans-serif;
  }
</style>

3. External CSS

External CSS is the most efficient way to style your web pages, especially when you have multiple pages. It involves linking an external CSS file to your HTML document using the <link> element.

<link rel="stylesheet" href="styles.css">

In this method, all styling rules are kept in a separate file, making your HTML cleaner and more manageable. It also allows you to reuse the same stylesheet across multiple HTML pages.

7-1. CSS Colors, Fonts, and Sizes

Here, we will demonstrate some commonly used CSS properties. You will learn more about them later.

1. CSS Color

The CSS color property defines the text color to be used. You can specify color in several ways: by name, HEX, RGB, RGBA, HSL, and HSLA.

  • Color Names: Common colors like red, blue, green, etc.
  • HEX: A 6-digit hexadecimal code representing the color, e.g., #ff5733.
  • RGB: Defines colors in terms of their Red, Green, and Blue components, e.g., rgb(255, 99, 71).
  • RGBA: Like RGB but includes an alpha (opacity) value, e.g., rgba(255, 99, 71, 0.5).
  • HSL: Defines colors in terms of Hue, Saturation, and Lightness, e.g., hsl(9, 100%, 64%).
  • HSLA: Like HSL but includes an alpha value for opacity, e.g., hsla(9, 100%, 64%, 0.5).
<p style="color: #ff0000;">This is a text with a HEX color.</p>

This is a text with a HEX color.

2. CSS Font Family

The CSS font-family property defines the font to be used for text. You can specify a specific font or a generic font family.

  • Specific Fonts: e.g., Arial, Verdana, Times New Roman.
  • Generic Font Families: Categories like serif, sans-serif, monospace, etc.
  • It's common to specify a list of fonts, allowing the browser to fall back to another if the first is unavailable.
<p style="font-family: 'Arial', sans-serif;">This text uses the Arial font.</p>

This text uses the Arial font.

3. CSS Font Size

The CSS font-size property defines the text size to be used. You can set the font size using various units like pixels, em, rem, percentages, etc.

  • Pixels (px): A fixed unit for setting font size, e.g., font-size: 16px;.
  • Percentage (%): A relative unit based on the parent element's font size, e.g., font-size: 120%;.
<p style="font-size: 20px;">This is a text with a font size of 20px.</p>

This is a text with a font size of 20px.

<p style="font-size: 200%;">This text uses a relative font size of 200%.</p>

This text uses a relative font size of 200%.

These three properties (color, font-family, and font-size) are fundamental to styling text in HTML. They allow you to create readable and visually appealing text on your web pages.
